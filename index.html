Keyboard Shortcuts

Use the following keyboard shortcuts to navigate:

  • Select/unselect seat: Enter or Space
  • Move to the next row: Alt + Page Down
  • Move to the previous row: Alt + Page Up
  • Move to the Continue button Ctrl + Alt + c
  • Open the Keyboard Shortcuts: Ctrl + ?
  • Close the Keyboard Shortcuts: Escape
